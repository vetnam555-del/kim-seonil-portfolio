HOW I WORK

다음 판단의 기준으로 남깁니다.

  1. 01문제를 분리한다

    매체·소재·타겟·랜딩·재고 중 실제 병목이 어디인지 구간별로 확인합니다.

  2. 02근거를 수집한다

    GA4와 매체 데이터를 교차 검증하고, 추정과 확인을 구분해 기록합니다.

  3. 03실행한다

  4. 04검증한다

    결과를 숫자로 판정합니다. 미달이면 원인을 분해하고, 실패도 기록에 남깁니다.

  5. 05기준으로 남긴다

    검증된 판단을 룰북과 자동화로 바꿔, 같은 성과가 반복되게 만듭니다.

J.ESTINA | IMC PERFORMANCE

검색에 갇힌 매출을
퍼널 전체로 열었습니다

소속

기여도
70%
주요 업무
IMC 전략 · 분석 체계 설계 · SA/DA 운영

이 사례가 증명하는 것 — 검색 의존 구조를 데이터로 진단하고, 고객 단계별 퍼널로 예산을 재설계할 수 있습니다.

  1. ①문제 정의

    매출은 나오는데, 성장이 멈췄습니다.

    검색·브랜드검색 중심의 운영은 효율이 좋았습니다. 문제는 그 효율이 이미 브랜드를 아는 사람에게서만 나온다는 것이었습니다. 신규 고객과 재구매 고객이 같은 메시지를 보고 있었고, 존재하는 수요를 거두는 것만으로는 성장 폭이 제한적이었습니다.

  2. ②분석 근거

    고객을 나누자, 채널의 역할이 보였습니다.

    GA4와 매체 데이터를 교차해 재구매 고객과 신규 유입 고객을 분리했습니다. SA·브랜드검색은 재구매 고객을 다시 데려오는 데는 강했지만 신규 수요 창출 기여는 낮았고, 인지 단계를 맡을 채널이 비어 있었습니다.

    개선 전
    인지담당 채널 없음
    탐색SA
    전환SA · 브랜드검색
    →
    개선 후
    인지DA (GFA·메타) — 신규 고객 메시지
    탐색쇼핑 · SA — 수요 연결
    전환브랜드검색 · SA — 재구매 전환
  3. ③실행

    채널이 아니라 고객 단계에 예산을 배분했습니다.

    • 인지 단계에 DA를 배치하고, 고객군별로 소재 메시지를 분리했습니다.
    • UTM Builder를 만들어 채널·소재·랜딩의 분석 기준을 통일했습니다.
    • SA는 풀퍼널 모니터링 체계(키워드 확장 → 미노출 관리 → T&D·랜딩 점검)로 재정비했습니다.
    • 매체·GA 교차 리포팅을 자동화해 보고 시간을 60% 줄였습니다.

    실행 구간 지표 — 검색 CPC −21% · CTR +7.3%

  4. ④결과

    GA ROAS 210% → 583%. 검증 가능한 숫자로 남겼습니다.

    검증

    GA 전환매출 약 58.3억 원(GA4 기준) 확인 — ‘검색 효율 관리’에서 ‘통합 퍼널 운영’으로 역할이 확장됐습니다.

  5. ⑤나의 역할

    IMC 재설계와 분석 체계 구축을 주도했습니다. 기여도 70%

    고객 세그먼트 정의, 채널 역할 설계, UTM 표준화, 성과 검증 체계까지 직접 설계했습니다. 집행 운영은 팀과 분담했습니다.

    LESSON 채널 효율로 싸우면 예산 이동의 근거가 없습니다. 고객 단계로 나누는 순간, 모든 예산 결정에 이유가 생겼습니다.

Source — 캠페인 마감 보고서, Weekly Report, GA4·매체 리포트. 공개 화면은 비식별 처리했습니다.

NEW BALANCE | FULL-FUNNEL × AUTOMATION

월 10억 규모의 운영을
자동화 위에 올렸습니다

소속 · 기간

운영 규모
월 10억 · 10개 채널
기여도
운영 40% · 자동화 설계 단독
주요 업무
통합 매체 운영 · 자동화 기획/구축

이 사례가 증명하는 것 — 10개 채널을 통합 운영하고, 그 운영을 지탱하는 자동화 시스템을 직접 설계·구축할 수 있습니다.

  1. ①문제 정의

    좋은 ROAS 뒤에서, 품절 상품이 광고비를 태우고 있었습니다.

    네이버 SA·BSA·GFA·쇼핑, 구글, 메타, 카카오, 크리테오, RTB하우스, 페이코 — 10개 채널과 시즌마다 바뀌는 수백 개 SKU. 재고·소진·소재 상태를 수기로 확인하는 데 하루 약 2시간이 들었고, 확인이 늦으면 품절 상품이 그대로 광고에 노출됐습니다.

  2. ②분석 근거

    판단하기 전에, 데이터부터 의심했습니다.

    • 운영 중 진행한 데이터 신뢰성 감사 — 리포트와 재산출 수치의 17.6% 격차를 분해해 그중 98.5%가 수집 지연이 아닌 매칭 범위 차이임을 규명하고, “D·D-1은 잠정치, D-2에 확정” 리포팅 기준을 세웠습니다.
    • 매체 ROAS만으로는 예산을 판단할 수 없다는 결론 — 재고 상태·검색 수요·매체 역할을 같은 화면에서 봐야 안전한 예산 이동이 가능했습니다.
    • 매체 성격별 판단 기준을 룰북으로 성문화했습니다. 예 — “브랜드검색은 ROAS가 높다는 이유로 증액을 제안하지 않는다.”
  3. ③실행

    확인 업무를 시스템에 넘겼습니다.

    재고 데이터1시간 주기 수집
    자동 판정≤10개 OFF · ≥20개 재라이브
    매체 API 실행회당 최대 900개 상품
    기록·검증시트 갱신 + 사후 QA
    알림조치 필요할 때만 메일
    • 리포트 자동화 — 데일리 리포트는 60분에서 5분으로, 위클리 리포트는 RAW 3종만 붙여넣으면 대시보드·QC까지 자동 생성됩니다.
    • 데드랜딩·소진 흐름 무인 모니터링, LG CNS MOP 기반 쇼핑검색 입찰 최적화 병행.
    • 안전장치 우선 설계 — 재고 데이터 이상(수집 실패율 급증) 시 자동 조치를 멈추고 사람을 호출합니다.
  4. ④결과

    확인하던 시간이, 판단하는 시간이 됐습니다.

    검증

    1회 QA에서 재고 소진 임계(1개 이하) 상태로 켜져 있던 광고 662건을 자동 감지·차단. 이 운영 방식은 LG CNS MOP 공식 도입 성공 사례로 소개됐습니다.

  5. ⑤나의 역할

    운영은 팀과 함께, 자동화는 단독으로. 운영 기여도 40%

    10개 채널 통합 운영은 팀 체제로, 재고 자동화·리포트 엔진·모니터링 시스템은 기획부터 구축·운영까지 단독으로 수행했습니다. 모든 시스템은 인수인계 패키지로 문서화되어 있습니다.

    LESSON 자동화의 핵심은 실행이 아니라 안전장치였습니다. ‘잘 돌아가는 자동화’보다 ‘이상하면 스스로 멈추는 자동화’가 운영을 지킵니다.

Source — 자동화 운영 로그, 매체 API 수집 데이터, 운영 시간 산정 기준. 브랜드 매출 등 민감 수치는 게재하지 않았습니다.

  1. ①문제 정의

    캠페인이 끝나면, 아무것도 남지 않았습니다.

    프리미엄 가전은 구매 전 정보 탐색이 긴 고관여 제품입니다. 그런데 단기 세일즈 광고는 집행이 끝나는 순간 소재도 트래픽도 사라졌습니다. 제품을 이해시키는 채널과 구매가 일어나는 접점이 분리되어 있었습니다.

  2. ②분석 근거

    시청 데이터가 ‘경험 콘텐츠’의 손을 들어줬습니다.

    시청–구독–구매 여정 데이터에서 제품 경험형 콘텐츠가 VTR 9.46%, CPA 950원을 기록했습니다. 구매 직전의 정보 탐색이 YouTube에서 일어나고 있었습니다 — 광고를 여정의 일부로 설계할 근거였습니다.

    경험형 영상 시청→채널 구독→라이브커머스→구매 전환
  3. ③실행

    콘텐츠–구독–커머스를 한 흐름으로 연결했습니다.

    • 제품 경험형 영상을 제작해 YouTube VAC로 증폭 — 캠페인 노출 1.4억 회.
    • 시청이 구독으로 이어지도록 채널 구독 유도 장치를 설계했습니다.
    • 모인 구독자를 라이브커머스로 초대해 체험–질문–구매가 한자리에서 일어나게 했습니다.
  4. ④결과

    구독자 2,000 → 100,600. 광고가 끝나도 채널은 남았습니다.

    검증

    라이브커머스 구매 전환율 12.27% — 단기 매출과 10만 구독 채널 자산을 동시에 확보. 광고비가 사라지는 비용이 아니라 쌓이는 자산이 됐습니다.

  5. ⑤나의 역할

    콘텐츠 커머스 전략을 설계했습니다. 기여도 70%

    여정 설계, 캠페인 구조(VAC·라이브커머스 연결), 성과 검증 체계를 담당했습니다. 콘텐츠 제작은 제작 파트와 협업했습니다.

    LESSON 콘텐츠는 소재가 아니라 자산입니다. ‘집행이 끝나도 남는가’를 기준에 넣자 캠페인의 구조 자체가 달라졌습니다.

Source — YouTube 캠페인 리포트(화면은 일부 캠페인 구간 캡처), 라이브커머스 성과 데이터. 노출 1.4억은 캠페인 전체 합계 기준입니다.
