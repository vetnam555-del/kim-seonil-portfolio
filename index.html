01

About

잘 만들어진 브랜드를, 더 잘 알리는 방법을 고민합니다

빠르게 변하는 트렌드에 발맞추는 동시에 고객의 이야기에 귀 기울여, 오래도록 가치 있는 소비 경험을 설계하는 마케팅을 지향합니다. 좋은 전략은 감이 아니라 데이터에서 나온다고 믿습니다.

디자인과 경영을 전공하고 창업·공모전으로 ‘만들고 알리는’ 전 과정을 직접 경험하며 30여 회 수상했습니다. 이후 KT그룹 플레이디에서 데이터 마케팅 PM으로 20여 개 브랜드의 퍼포먼스를 책임지며 검색·디스플레이·SNS를 통합한 IMC 운영과 GA4·SQL 기반 분석을 단련했고, 현재 중앙그룹 HLL중앙 퍼포먼스전략국 과장으로 퍼포먼스와 브랜드를 아우르는 디지털 마케팅 전략을 총괄하고 있습니다.

저의 차별점은 ‘양면의 시야’입니다. 콘텐츠 하우스인 HLL중앙에서 매체를 파는 쪽과 광고를 사는 쪽의 논리를 모두 이해합니다. 월 10억~15억 규모 미디어 예산을 운용하고 외부 대행사를 선정·관리하며, 매체 효율과 광고주 성과의 균형점을 찾습니다.

데이터 분석

GA4·SQL·UTM Builder로 전환 경로를 진단하고, 세그먼트별 성과를 리포팅까지 자동화합니다.

IMC 통합 운영

검색·디스플레이·SNS를 하나의 메시지로 잇는 미디어믹스를 설계해 인지부터 전환까지 연결합니다.

전략 · 자동화

CRM과 마케팅 자동화(MOP)로 고객 여정 전반을 설계하고, 예산·입찰을 실시간으로 최적화합니다.

02

How I Work

감으로 시작하지 않고, 데이터로 진단한 뒤 실행과 개선까지 연결합니다.

  1. STEP 01

    진단

    GA4, 매체 데이터, 검색 수요, 전환 경로를 함께 보며 현재 병목이 소재·매체·랜딩 중 어디에 있는지 분리합니다.

    GA4 · RAW 데이터 · 경쟁사 벤치마크
  2. STEP 02

    설계

    KPI를 다시 정리하고 타겟, 메시지, 미디어믹스를 하나의 퍼널 흐름으로 묶어 실행 구조를 만듭니다.

    KPI 설계 · 미디어믹스 · 세그먼트
  3. STEP 03

    운영

    소재, 예산, 입찰, 지면을 반복 조정하며 전환 효율을 관리하고, 자동화 가능한 업무는 운영 리스크를 줄입니다.

    A/B 테스트 · 자동입찰 · API 자동화
  4. STEP 04

    증명

    성과를 단순 수치가 아니라 원인, 실행 내용, 다음 개선안까지 연결해 리포트와 의사결정 자료로 남깁니다.

    리포트 자동화 · 인사이트 · 개선안
